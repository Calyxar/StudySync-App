# StudySync

A clean, light-theme productivity app built with **Expo (React Native)** to help users stay consistent with tasks, focus sessions, and progress tracking.

## Current Features (in progress)
- Dashboard (today’s tasks + weekly overview)
- Tasks (create / edit / details)
- Focus session timer
- Analytics & streak tracking
- Settings (notifications, appearance, account)

## Tech Stack
- Expo + React Native
- TypeScript
- expo-router
- Local storage / backend integration (planned) *(update if different)*
